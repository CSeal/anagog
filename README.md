## Getting Started

0. Create `.env` file from example
```sh
$ cp .env.example .env
```

2. Run Docker
```sh
$ yarn docker
```

3. Run back-end server(init db and create admin user - login: admin, password: Anagog22)
```sh
$ yarn server
```

4. Run front-end React
```sh
$ yarn start
```
